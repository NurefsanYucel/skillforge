# SkillForge

SkillForge is a full-stack learning tracker built with Next.js, TypeScript, Tailwind CSS, and Supabase.

## Features

- User authentication
- Protected dashboard
- Create, edit, delete goals
- Add learning resources to goals
- Track progress
- Search goals
- Profile page
- Profile picture upload
- Recent activity feed

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Supabase Auth
- Supabase PostgreSQL
- Supabase Storage

## Setup

```bash
npm install
npm run dev
